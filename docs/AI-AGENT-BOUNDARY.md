# AI Agent Boundary Policy

**Status:** Active
**Last Updated:** 2026-07-17

## Purpose
This document defines repository boundaries for AI-agent behavior and deployment-related documentation.

## Policy
- This repository does **not** adopt unsupervised, in-character autonomous social-posting workflows.
- Character and narrative content (including Violet) is maintained as authored game/lore documentation.
- Any future automation proposals must include explicit human-review controls before publish and be reviewed as governance changes before adoption.

## Scope
Applies to active specifications, docs, and implementation plans in this repository.
