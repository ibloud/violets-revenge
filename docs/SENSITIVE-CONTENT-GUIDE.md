# Sensitive Content Guide

Core principle: assault is implied through forensic aftermath, never depicted on-screen.

## Do
- Use clinical/procedural language.
- Keep sensitive references contextual and restrained.
- Use content warnings in public-facing material.

## Don't
- Depict assault in gameplay/cinematics/marketing.
- Add gratuitous violence or shock framing.
- Humanize or identify Guilty archetypes.
